# Portfolio v2 - Paul Manoha

Bienvenue sur le dépôt de mon portfolio personnel. Ce projet est une vitrine de mes compétences dans le cadre de mon BTS SIO.

> **Live Demo :** https://astropathe.github.io/portfoliov2/

### Fonctionnalités :
* **Terminal Interactif :** Supporte les commandes `help`, `about`, `skills`, `projects`, `clear`.
* **Design Responsive :** Thème sombre (Cyber-look) harmonisé avec les couleurs de Parrot OS / Debian.
* **Architecture Propre :** Séparation HTML/CSS/JS et organisation par répertoires (`/pages`, `/photo`).

## Stack Technique
* **Frontend :** HTML5, CSS3 (Flexbox & Grid), JavaScript (ES6+).
* **OS de développement :** Parrot OS 🦜
* **Versionning :** Git & GitHub.
* **Hébergement :** GitHub Pages.
